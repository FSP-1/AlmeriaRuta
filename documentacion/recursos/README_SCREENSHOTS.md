Carpeta de capturas de pantalla para la documentacion.

Instrucciones:
- Añade imagenes en formato PNG o JPG usando los nombres sugeridos abajo.
- Mantén resolucion aproximada 1080x1920 para pantallas moviles o 1280x720 para vistas web.

- home_card_line.png           # Tarjeta de linea en Home

- map_filters_panel.png        # Panel de filtros en el mapa
- map_stop_sheet.png           # Ficha de parada / punto turistico
- map_navigation_view.png      # Vista de navegacion (polilineas)
- map_walk_bus_walk.png        # Polilineas separadas: caminar/bus/caminar

- lines_list.png               # Listado de lineas
- line_detail.png              # Detalle de linea con paradas

- ticket_selection.png         # Pantalla de seleccion de ticket
- ticket_confirmation.png      # Confirmacion de compra y ticket generado

- validation_scan.png          # Vista de escaneo/validacion (lector)
- validation_result.png        # Resultado de validacion (valido/duplicado)

- cards_list.png               # Listado de tarjetas/estado
- recharge_dialog.png          # Dialogo de recarga y comprobante

- search_field_usage.png       # Campo de busqueda en distintas vistas
- favorites_panel.png          # Panel de favoritos (paradas/lineas)
